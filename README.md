 jquery rest form plugin
=======================

## About
 jQuery Form PluginをRESTfulなURLに対応させるためのラッパープラグインです。

## jQuery Form Plugin
http://malsup.com/jquery/form/

## Required
* jQuery(Recommende higher version)
* jQuery Form Plugin

## Usage
 urlに[:(param_name)]と入れると該当パラメータに置換します。
 ex: /user/:user_name -> /user/hoge
